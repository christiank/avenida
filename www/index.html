Avenida

Avenida is a multimedia scripting runtime environment. It is a suite of tools which allow for rapid prototyping of digital media operations.

What's Included

  • The avenida command line interpreter

System Requirements

  • GraphicsMagick
  • BSD Make
  • A C11 compiler (anything with support for anonymous unions) — Clang is fine

Avnscript Reference

Avnscript is a miniature programming language geared towards multimedia. It actually is nothing but a tricked-out Lua interpreter with capabilities to query and manipulate multimedia files. Performing batch operations on digital media files with Avnscript is simple and straightforward.

Media Species

To avoid ambiguities with terms such as "format," Avnscript uses a different set of terms.

Avenida recognizes four media species: raster graphics, vector graphics, audio, and video. (To be perfectly clear, "video" has no sound, by definition. It's just video.)

How it Works

When you invoke a media operation, you're not actually applying that operation onto the media. Instead, Avenida makes a note of what would happen. Operations are not actually applied until you call the render() function.

This gives the interpreter the opportunity to verify problems. In a future release, the interpreter will also take this opportunity to strategize and find the fastest way to get the result based on the operations in the list.
